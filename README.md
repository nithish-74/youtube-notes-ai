# YouTube Notes AI

Paste a YouTube link — get clear **English notes** from any video.

Supports **Hindi, Telugu, Tamil**, and more (auto-translated to English).

## Author

**[Nithish Reddy](https://github.com/nithish-74)** ([@nithish-74](https://github.com/nithish-74))

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
